# Ressources — données de liaison HubSpot

Ce dossier ne contient pas de contenu éditorial : le blog reste géré et publié sur HubSpot / meandyoutoo.app, afin d'éviter toute duplication de contenu SEO.

`articles-hubspot.json` liste, par thématique D&I, l'URL vers laquelle la page `ressources.html` doit rediriger. Chaque URL est un placeholder (`https://www.meandyoutoo.app/`) à remplacer par le lien réel de la catégorie ou de l'article correspondant sur le blog HubSpot avant mise en production.

Ce fichier n'est pas encore consommé dynamiquement par `ressources.html` (V1 front en HTML statique) : il sert de source de vérité pour les liens, à connecter lors de l'intégration backend.
