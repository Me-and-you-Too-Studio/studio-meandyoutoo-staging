# Studio Me&YouToo — Front staging

Front HTML/CSS/JS statique pour tester la direction graphique du Studio Me&YouToo.

## Structure

- `index.html` : accueil / dashboard
- `mes-campagnes.html` : liste des campagnes
- `bibliotheque.html` : exploration des sujets
- `theme-management.html` : détail d’une thématique
- `composer.html` : composition d’une campagne
- `campagne-detail.html` : pilotage d’une campagne
- `ressources.html` : liens vers ressources HubSpot
- `assets/css/style.css` : design system centralisé
- `assets/img/` : logos, favicon, illustrations SVG
- `assets/js/app.js` : interactions front simples
